Improve Performance

A solution that reduces the document load time from 12 seconds to 2 seconds

With server hosted in Singapore and new users accessing from New York and Hong Kong, these remote users experienced low latency. Loading a document took up to 12 seconds.

• Solution implemented
1. Caching servers were deployed to New York and Hong Kong. Document accessed were cached in the respective cache server so that subsequent access would be loaded from the cache. However document accessed for the first time still experienced slow loading.
2. A scheduled service to pre-cache new documents was developed and deployed. Time to load a document for the first time reduce to 2 seconds.
